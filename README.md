# Getting Started
## Prerequisites
- npm
```node
npm init [-y]
```

- Make sure both ports 3000 and 3001 are open
## Installation
- Inside the root of the project and then again inside the client directory run
```node
npm install
```
## Running the project
- Open a new terminal window inside the root of the project and run
```node
npm start
```
